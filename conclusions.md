## Ключові аналітичні висновки за результатами запитів

- Реєстрації рівномірно розподілені в період січень–квітень 2024, без різких сезонних піків.

- Демографія студентів \
Найбільша кількість студентів — з Києва, що може свідчити про потенційно вищий попит у великих містах.

- Популярність курсів і категорій \
Категорії Data Science, Machine Learning та Programming мають найбільшу кількість курсів.

- Курси з UI/UX та Machine Learning мають найвищі середні оцінки, що може вказуквати на високу якість контенту та відповідність очікуванням студентів.

- Успішність та прогрес студентів \
Середній бал більшості студентів знаходиться в діапазоні 85–95, що свідчить про високу загальну успішність.
Є курси з незавершеними уроками, зокрема у програмуванні (Python, SQL для початківців), що може свідчити про те, що частина студентів стикається зі складністю матеріалу або втрачає мотивацію на середині курсу.

- Топ-курси в категоріях \
Для кожної категорії визначено лідерів за кількістю студентів, що може бути використано:
у маркетингу,
для формування рекомендацій,
як еталон для інших курсів.

## Рекомендації з покращення бази даних

Додати інформацію:

- загальну кількість уроків у курсі
- тривалість курсу
- складність (beginner / intermediate / advanced)
- у таблиці progress фіксувати дату проходження уроку
- відстежувати ввідтік студентів

## Бізнес-рекомендації

- Посилити маркетинг курсів-лідерів (SQL, Python)
- Оптимізувати складні уроки з низьким completion rate
- Запровадити персональні рекомендації курсів
- Використовувати рейтинг студентів для гейміфікації
